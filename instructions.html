This is a demo banking website made using HTML, CSS and Javascript.

There is no data base implemented in this project, so you cannot make an account and/or save changes in the transaction movements. Every time you reload the webpage, all the data will reset.

You can transfer money between accounts, take a loan and delete accounts.

The username of any user is the initials of that user in lowercase and without spaces. So, for example, the username of the user 'Vedang Danej' will be 'vd'.

To transfer between accounts, enter the username of the user (initials).

The maximum amount of loan given by the bank is ten times the maximum deposit made previously. This can be used to get an infinite amount of loan from the bank (or at least as much as 'let' can safely store).

Again, to delete an account, use the username and not the name of the user.

These people have accounts in the bank.
